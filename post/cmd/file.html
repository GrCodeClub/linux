    pwd

    cp

    mv

    Διαγράφει αρχεία ή καταλόγους. (Προσοχή: Μόνο με προσεκτική χρήση!)

    rm

    find

    Τροποποιεί τα δικαιώματα πρόσβασης σε αρχεία και καταλόγους.

    chmod [file_name]

    Execute permission

    chmod +x [file_name]

    Read permission

    chmod +r [file_name]

    Write permission

    chmod +w

    Read, write and execute permissions to the file owner:

    chmod u+rwx [file_name]

    Read, write and execute permissions to the file group:

    chmod g+rwx [file_name]

    Read, write and execute permissions to the file All (owner,groups,others):

    chmod a+rwx [file_name]

    Read, write and execute permissions to the file Others:

    chmod o+rwx [file_name]

    Remove write permission for the group:

    chmod g-w [file_name]

    Αλλάζει τον κάτοχο ενός αρχείου ή καταλόγου.

    chown
