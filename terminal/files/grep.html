    Εντολή grep

    Χρησιμοποιώντας την παρακάτω εντολή θα μας εμφανίσει όλες τις γραμμές του αρχείου file.txt που περιέχουν την λέξη grcodeclub.

    grep grcodeclub file.txt

    Αναζήτηση σε πολλά αρχεία 

    grep grcodeclub file1.txt file2.txt file3.txt

    Αναζήτηση σε όλα τα αρχεία που βρίσκονται στο τρέχον φάκελο.

    grep grcodeclub *

    Για να βρείτε μόνο ολόκληρες λέξεις.

    grep -w grcodeclub file.txt

    Για να εμφανίζει όλες τις γραμμές που εμφανίζεται η λέξη ανεξάρτητος αν είναι κεφάλαια ή πεζά.

    grep -i grcodeclub file.txt

    Για να εμφανίσουμε όλες τις γραμμές που δεν περιέχουν το μοτίβο/λέξη που δώσαμε.

    grep -v grcodeclub file1.txt

    Για να δούμε σε ποια αρχεία από το τρέχον φάκελο περιχέουν την λέξη που δώσαμε.

    grep -l grcodeclub *
