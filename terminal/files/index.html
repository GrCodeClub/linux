    Διαγραφή

    Διαγράφει αρχεία ή καταλόγους. (Προσοχή: Μόνο με προσεκτική χρήση!)

    rm

    Το -f (--force) διαγράφει χωρίς ερώτηση

    rm -f

    Επιβεβαίωση διαγραφής από τον χρήστη 

    rm -i

    Διαγράφει άδειους καταλόγους

    rm -d

    Διαγράφει όλα τα αρχεία και τους καταλόγους αναδρομικά

    rm -r
    rm -R

    Εντολή ls

    Εμφανίζει τα περιεχόμενα ενός καταλόγου.

    ls

    Το -l ενός επιπρόσθετες πληροφορίες για κάθε αρχείο.

    ls -l

    Το -a (--all) εμφανίζει και τα αρχεία που το όνομά τους αρχίζει με τελεία (κρυφά αρχεία)

    ls -a

    ls -t

    ls -g

    Το -r (--reverse) εμφανίζει τα αρχεία με αντίστροφη σειρά ταξινόμησης

    ls -r

    ls -i

    ls -d

    Το -R εμφανίζει αναδρομικά τα περιεχόμενα ενός καταλόγου και των υποκαταλόγων του

    ls -R

    ls -S

    Δικαιώματα

    Τροποποιεί τα δικαιώματα πρόσβασης σε αρχεία και καταλόγους.

    chmod [file_name]

    Execute permission

    chmod +x [file_name]

    Read permission

    chmod +r [file_name]

    Write permission

    chmod +w

    Read, write and execute permissions to the file owner:

    chmod u+rwx [file_name]

    Read, write and execute permissions to the file group:

    chmod g+rwx [file_name]

    Read, write and execute permissions to the file All (owner,groups,others):

    chmod a+rwx [file_name]

    Read, write and execute permissions to the file Others:

    chmod o+rwx [file_name]

    Remove write permission for the group:

    chmod g-w [file_name]

    Αλλάζει τον κάτοχο ενός αρχείου ή καταλόγου.

    chown
